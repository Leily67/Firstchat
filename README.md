## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```

## Deployment

To deploy this project run

```bash
  cd my_project
  cd backend
  npm install
  npm run start
```

```bash
  cd my_project
  cd frontend
  npm install
  npm run start
```

run project this : 127.0.0.1:8080
